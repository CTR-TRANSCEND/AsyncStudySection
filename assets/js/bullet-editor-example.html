AGR Bullet Editor - Example Usage

Note: This is a demonstration page for the AGR Bullet Editor module. Ensure all required dependencies are loaded before the editor initializes.

Example 1: Auto-initialization

Add data-bullet-editor="auto" to any textarea for automatic initialization:

<textarea name="strengths" data-bullet-editor="auto" rows="5"></textarea>

Example 2: Manual Initialization

Initialize editors programmatically with custom options:

const editorId = AGR.BulletEditor.init('#weaknesses', { maxLength: 2000, showCounter: true, enablePreview: true, autoSaveDelay: 5000 });

Example 3: Get/Set Content

Programmatically get and set editor content:

// Get content const bullets = AGR.BulletEditor.getContent(editorId); // Set content AGR.BulletEditor.setContent(editorId, [ 'First point', 'Second point', 'Third point' ]);
